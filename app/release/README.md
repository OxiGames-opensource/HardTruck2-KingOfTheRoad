# Release

This directory contains locally generated release packages.

Generated artifacts are intended for local testing and publication through GitHub Releases.

## Directory Structure

```text
Linux/
Windows/
```

## Contents

Generated release packages may include:

- OxiGames patcher;
- patch definitions;
- documentation;
- license information.

Original game files are never included.

## Repository Policy

Only placeholder files (such as `.gitkeep`) are tracked by Git.

Generated binaries and release archives must not be committed to the repository.
