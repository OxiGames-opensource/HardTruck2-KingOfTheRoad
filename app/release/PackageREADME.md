# OxiGames Hard Truck 2 Patcher

This package contains compatibility patches for **Hard Truck 2: King of the Road**.

The patcher is intended for use with legally obtained copies of the game.

## Contents

```text
OxiGames-HardTruck2-Patcher
patches/
README.md
LICENSE
```

## Usage

1. Extract the archive.
2. Run **OxiGames-HardTruck2-Patcher**.
3. Click **Browse**.
4. Select `king.exe` or the game directory.
5. Choose the desired patch state.
6. Click **Apply changes**.
7. Start the game.

## Included Patches

* Display Mode Detection
* Texture Memory

## Restore Original File

The patcher automatically creates a backup before the first modification.

Use **Restore original** to restore the original executable from the backup.
